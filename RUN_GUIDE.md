## 1. 运行环境要求

- **后端**: Python 3.10+ (建议使用 Conda 环境)
- **前端**: Node.js 18+ (推荐使用 npm 或 pnpm)
- **数据库**: SQLite (默认内置)
- **AI 引擎**: 需要配置 `.env` 文件中的 LLM API Key (支持 OpenAI 格式)

---

## 2. 启动说明 (前后端分别启动)

### 2.1 后端服务 (Backend)

后端采用 FastAPI 框架，请按照以下步骤启动：

1. **环境激活** (可选):
   确保您已进入虚拟环境（如 Conda 或 venv）。

2. **数据库初始化**:
   如果是首次运行，或需要重置测试数据，请在项目根目录下执行：
   ```bash
   python src/database/init_db.py
   ```

3. **启动 FastAPI 服务**:
   在项目根目录下运行以下命令（默认端口 8000）：
   ```bash
   python -m uvicorn src.web.app:app --host 127.0.0.1 --port 8000 --reload
   ```
   *服务启动后，可通过 `http://127.0.0.1:8000/docs` 查看 API 文档。*

### 2.2 前端服务 (Frontend)

前端采用 Vue 3 + Vite 架构，请按照以下步骤启动：

1. **进入前端目录**:
   ```bash
   cd pet-frontend
   ```

2. **安装依赖** (首次运行需执行):
   ```bash
   npm install
   ```

3. **启动开发服务器**:
   ```bash
   npm run dev
   ```
   *访问浏览器：`http://localhost:5173`。前端已配置代理，将自动转发 `/api` 请求至后端 8000 端口。*

## 3. 核心功能演示路径 (Thesis Highlights)

### 3.1 演示场景 A：领养人渐进式申请 (User A)
- **登录账号**: `adopter_a@test.com` / `123456`
- **操作步骤**:
  1. 进入 **[领养中心]**。
  2. 使用顶部的 **[地区筛选]**（如选择：杭州）锁定本地宠物。
  3. 点击宠物 **[团团]**，右侧将展开精美的“宠物档案”，首先阅读其性格故事与领养约束。
  4. 点击 **[开始 AI 领养评估]**。
  5. 体验三步式表单：从语义化动机到量化能力（预算/时间）的录入。
  6. 提交后，前往 **[个人中心]** 观察申请状态从 `evaluating` 变为 `waiting_publisher`。
  7. 点击申请详情，查阅生成的 **多维度 AI 审计报告**、风险提示及 Trace ID。

### 3.2 演示场景 B：送养人协作决策 (User B)
- **登录账号**: `sender_b@test.com` / `123456`
- **操作步骤**:
  1. 进入 **[个人中心]** 下方的 **[收到的领养申请]**。
  2. 点击 User A 的申请项。
  3. **核心点**: 观察 AI 如何识别出领养人的潜在风险（如加班风险）并给出具体的 **建议追问项**。
  4. 在底部的 **[裁决工作台]** 中，选择“通过”或“发起追问”，并输入理由。
  5. 提交决策，观察系统如何记录人类反馈。

### 3.3 演示场景 C：后验学习闭环 (Feedback Loop)
- **操作步骤**:
  1. 当 User A 的申请被通过后，User A 登录并点击 **[确认接宠物回家]**。
  2. 状态流转至 `adopted`。
  3. User A 填写 **[回访反馈表单]**，提交 5 分好评。
  4. **学术深度点**: 系统后台会自动更新 `adoption_signal_weights` 表，根据本次成功案例，动态降低“加班风险”等标签在后续评估中的惩罚权重。

### 3.4 演示场景 D：管理员科研看板 (Dashboard)
- **操作步骤**:
  1. 以管理员账号登录（或直接访问 `/dashboard`）。
  2. 在 **[管理总览]** 中查看：
     - **人机决策一致率**: 衡量模型预测的准确度。
     - **多智能体博弈轨迹**: 实时滚动展示 Agent 之间的协商冲突。
     - **风险置信度雷达图**: 展示后验学习对不同维度风险判断的修正成果。
