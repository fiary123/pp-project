### 演示小技巧

1. **角色隔离演示**：
   - 使用 **无痕模式** 或不同的浏览器分别登录 `root@test.com` (管理员) 和 `user@test.com` (普通用户)。
   - 演示“用户提交申请 -> 管理员在可视化看板实时查收 -> 启动 AI 审核 -> 人工最终裁决”的完整闭环。
2. **可视化看板**：
   - 登录管理员账号后，点击“管理后台”，查看基于 ECharts 的实时发帖分布、活跃趋势图表。
3. **数据重置**：
   - 如需恢复初始干净的测试环境，请在终端执行：
     ```powershell
     python src/database/init_db.py
     ```
