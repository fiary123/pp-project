一、养宠通用原则（所有宠物适用）
责任优先：宠物寿命从几年到十几年，需长期投入时间、金钱与精力，不轻易弃养。
合规饲养：异宠 / 保护动物需确认是否合法，不购买、饲养野生保护动物。
科学喂养：拒绝 “人吃的都能喂”，严格遵循宠物专属饮食与禁忌。
健康管理：定期驱虫、疫苗、体检，出现异常及时就医。
环境安全：消除家中危险物品（电线、有毒植物、药品、小零件）。
二、伴侣宠物：猫 & 狗（最主流）
（一）猫咪饲养指南
1. 选宠与品种
新手友好：英短、美短、布偶（温顺）、橘猫（皮实）、暹罗（活泼）。
避坑：折耳猫（遗传病）、后院繁殖猫（健康风险高）。
2. 环境布置
必备：猫砂盆（1 猫 1 盆 + 1 备用）、猫抓板 / 柱、猫窝、食水碗、猫爬架。
猫砂：膨润土 / 豆腐砂，每日清理 1–2 次，每周彻底换砂 + 洗盆。
安全：封窗、收走百合（对猫剧毒）、电线套保护管。
3. 饮食（纯肉食动物，核心是高蛋白）
主食：
幼猫（≤12 月）：幼猫粮 + 主食罐，每日 4–5 餐，动物蛋白≥35%。
成猫：定时定量，湿粮占比≥50%（防尿结石），每日 2 餐。
禁忌：巧克力、葡萄 / 葡萄干、洋葱、大蒜、百合、木糖醇、生鱼 / 生肉（寄生虫）、高盐高油人类食物。
饮水：猫不爱喝水，用流动饮水机 + 冻干复水，每日换新鲜水。
4. 日常护理
梳毛：短毛每周 2–3 次，长毛每日 1 次，防毛球、掉毛。
洗澡：1–2 个月 1 次，用宠物专用香波，彻底吹干防皮肤病。
清洁：
耳朵：每周清洁外耳，黑褐色分泌物 / 异味需就医。
牙齿：每周刷牙 / 洁齿零食，防牙结石。
指甲：每 2–3 周剪 1 次，避开血线。
5. 健康与疫苗
疫苗：
猫三联（猫瘟、鼻支、杯状）：8 周首针，3 周 1 次，共 3 针；每年加强。
狂犬：3 月龄以上，每年 1 针。
驱虫：外驱每月 1 次，内驱每 3 个月 1 次。
常见问题：呕吐、软便、频繁上厕所、精神差，及时就医。
6. 互动与训练
猫独立，每日 10–15 分钟逗猫（逗猫棒、激光笔）即可。
训练：用零食奖励 “过来”“坐下”，禁止打骂。
（二）狗狗饲养指南
1. 选宠与品种
新手友好：金毛、拉布拉多、柯基、比熊、泰迪（小型）、柴犬。
按需求选：
小户型 / 上班族：比熊、泰迪（运动量小）。
有孩子 / 大空间：金毛、拉布拉多（温顺、耐造）。
爱运动：边牧、哈士奇（精力旺盛）。
2. 环境布置
必备：狗窝、食水碗、牵引绳、胸背带、狗厕所 / 尿垫、玩具（磨牙棒）。
安全：收走巧克力、葡萄、木糖醇、药品，阳台 / 楼梯防护。
3. 饮食（杂食，以肉为主）
主食：
幼犬（≤12 月）：幼犬粮，每日 3–4 餐，泡软喂食。
成犬：每日 2 餐，按体重 / 运动量调整，肥胖犬换低脂粮。
禁忌：巧克力、葡萄 / 葡萄干、洋葱、大蒜、木糖醇、骨头（易卡喉 / 划伤）、高盐高油人类食物。
饮水：随时新鲜水，夏季多补。
4. 日常护理
梳毛：短毛每周 3–4 次，长毛每日 1–2 次，防打结、掉毛。
洗澡：1–2 个月 1 次，用宠物香波，吹干防皮肤病。
清洁：
耳朵：每周清洁，异味 / 分泌物就医。
牙齿：每周刷牙 / 洁齿棒。
指甲：每月剪 1 次，避免过长扎肉。
遛狗：
小型犬：每日 1–2 次，每次 20–30 分钟。
中大型犬：每日 2–3 次，每次 40–60 分钟。
5. 健康与疫苗
疫苗：
联苗（犬瘟、细小、肝炎等）：6–8 周首针，3 周 1 次，共 3 针；每年加强。
狂犬：3 月龄以上，每年 1 针。
驱虫：外驱每月 1 次，内驱每 3 个月 1 次。
常见问题：腹泻、呕吐、咳嗽、跛行，及时就医。
6. 训练与社会化
基础指令：坐、卧、来、随行，用零食奖励，耐心重复。
社会化：幼犬期多接触人、狗、环境，减少胆小 / 攻击性。
三、小型啮齿类：仓鼠、金丝熊、龙猫、豚鼠
（一）仓鼠 / 金丝熊
1. 环境
笼子：基础笼≥40×30×30cm，配跑轮（直径≥17cm）、小窝、食盆、水壶。
垫料：木屑 / 纸棉，每周换 1 次，保持干燥。
温度：20–26℃，避免阳光直射、风口。
2. 饮食
主食：专用仓鼠粮，每日 1–2 次，量以吃完为宜。
辅食：少量面包虫干、蔬菜干、苹果、胡萝卜（每周 1–2 次）。
禁忌：人类零食、高糖高油、生水（用凉白开）。
3. 护理
独居（仓鼠领地意识强，合笼易打架）。
每周清理笼子，不水洗（易应激），用湿巾擦拭。
不强行抓握，从下方托抱。
（二）龙猫
1. 环境
笼子：高笼（垂直空间），配跳板、窝、食盆、水壶、草架。
温度：15–23℃（怕热，25℃以上易中暑），湿度≤60%。
垫料：木屑 / 纸粒，每周清理。
2. 饮食
主食：提摩西草（无限量）+ 龙猫粮（每日 1–2 次）。
辅食：少量苹果干、蒲公英干（每周 1–2 次）。
禁忌：高糖、高水分水果（如西瓜）、坚果、人类零食。
3. 护理
沙浴：专用浴沙，每周 2–3 次，清洁毛发。
牙齿：提供磨牙石 / 苹果枝，防牙齿过长。
（三）豚鼠（荷兰猪）
1. 环境
笼子：≥60×40×40cm，可群居（同性），配食盆、水壶、草架、厕所。
垫料：木屑 / 纸棉，每日清理厕所，每周换垫料。
温度：18–25℃，通风干燥。
2. 饮食（需补充维生素 C）
主食：提摩西草（无限量）+ 豚鼠粮（每日 1–2 次）。
辅食：新鲜蔬菜（青椒、生菜、胡萝卜），每日少量，补维 C。
禁忌：高糖水果、洋葱、大蒜、豆类。
3. 护理
可上手抚摸，性格温顺。
每周清理笼子，提供磨牙物品。
四、鸟类：鹦鹉（虎皮 / 玄凤 / 牡丹）、文鸟、金丝雀
（一）鹦鹉（新手首选：虎皮、玄凤、牡丹）
1. 环境
笼子：金属笼（防啃咬），尺寸≥40×30×40cm，配栖木、食盒、水盒、墨鱼骨（补钙磨嘴）。
位置：安静、通风、避免阳光直射 / 风口。
温度：15–28℃，冬季保暖。
2. 饮食
主食：混合谷物（谷子、黍子、加纳利子）。
辅食：每周 2 次新鲜蔬果（苹果、胡萝卜、青菜），墨鱼骨 / 钙粉补钙。
饮水：每日换凉白开。
3. 护理
笼具每周清理 1 次，粪便每日清理。
夏季可提供水浴盆，让鸟自行洗澡。
互动：每日 10–15 分钟，训练上手、吹口哨。
（二）文鸟 / 金丝雀
1. 环境
竹制 / 金属笼，空间适中，配栖木、食水盒。
安静环境，避免惊吓。
2. 饮食
主食：谷物 + 稗子 + 小米。
辅食：少量青菜、苹果，每周 1–2 次。
3. 护理
笼具清洁同鹦鹉，以观赏为主，互动较少。
五、爬行 / 异宠：鬃狮蜥、玉米蛇、角蛙（新手进阶）
（一）鬃狮蜥（温顺，新手友好）
1. 环境
饲养箱：≥60×40×40cm，配加热灯、UVB 灯、晒台、躲避穴。
温度：晒点 30–35℃，冷区 24–28℃；夜间 22–25℃。
垫材：爬沙 / 报纸，每周清理。
2. 饮食
幼体：每日 1 次，蟋蟀、杜比亚蟑螂（昆虫为主）。
成体：隔日 1 次，昆虫 + 蔬菜（油麦菜、南瓜、胡萝卜）。
补钙：每周 1–2 次钙粉 + D3，撒在食物上。
3. 护理
每日清洁粪便，每周换垫料。
可上手，性格温顺，不咬人。
（二）玉米蛇（无毒，安静）
1. 环境
饲养箱：≥40×30×30cm，配加热垫、躲避穴、水盆。
温度：26–28℃，热点 28–30℃；避光。
垫材：木屑 / 厨房纸，每周清理。
2. 饮食
主食：小白鼠（乳鼠 / 亚成体），每周 1 次，按体型选大小。
喂食后 24 小时不打扰，避免吐食。
3. 护理
安静饲养，少上手，避免应激。
蜕皮期提供湿润环境，不强行撕皮。
（三）角蛙（两栖，新手友好）
1. 环境
饲养盒：≥20×15×15cm，分水区（浅水）+ 陆区（湿苔藓 / 椰土）。
温度：22–28℃，湿度 70%–80%。
水质：用晒过的自来水 / 纯净水，每日换水。
2. 饮食
主食：蟋蟀、杜比亚、小鱼，每周 2–3 次。
禁忌：喂过大食物，避免消化不良。
3. 护理
每周清理饲养盒，保持水质清洁。
少上手，皮肤敏感易受伤。
六、水族：金鱼、热带鱼（孔雀鱼、斑马鱼）
1. 环境
鱼缸：新手选≥30cm 缸，配过滤、加热棒（热带鱼）、灯、造景。
水质：
金鱼：冷水，pH7.0–8.0，每周换 1/3 水。
热带鱼：24–28℃，pH6.5–7.5，每周换 1/4 水。
养水：新缸开过滤 3–7 天，建立硝化系统再放鱼。
2. 饮食
主食：专用鱼粮（颗粒 / 薄片），每日 1–2 次，5 分钟内吃完。
辅食：丰年虾、红虫（每周 1–2 次）。
禁忌：过量喂食（坏水）、变质食物。
3. 护理
每日观察鱼状态（游动、食欲、鳃部）。
每周清洗过滤棉，每月清洗滤材（保留部分旧滤材）。
七、养宠常见误区（避坑）
误区：宠物吃人类食物更健康 → 正解：人类食物高盐高油，易致中毒 / 疾病。
误区：洗澡越勤越干净 → 正解：猫狗 1–2 个月 1 次，频繁洗伤皮肤。
误区：异宠不用驱虫 / 疫苗 → 正解：爬宠 / 啮齿类也需定期驱虫，防寄生虫。
误区：笼养不用互动 → 正解：多数宠物需陪伴，长期孤独易抑郁 / 行为异常。
